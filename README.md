# Reserva Plus

Plugin GLPI para reservas com painel moderno, aprovações, regras, bloqueios, calendário, notificações e relatórios.

## Escopo inicial

- Plugin instalavel sem alterar o core do GLPI.
- Menu em Ferramentas.
- Permissões por perfil.
- Tabelas proprias do plugin.
- Painel e páginas iniciais para reservas, calendário, aprovações, regras, bloqueios, relatórios e configurações.

## Requisitos

- GLPI 11.0.x.
- PHP 8.2 ou superior, preparado para PHP 8.5.

## Nome técnico

O nome técnico é `reservaplus`, sem hífen, para manter compatibilidade com o padrão de funções do GLPI.
